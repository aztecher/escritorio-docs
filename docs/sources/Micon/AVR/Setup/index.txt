==============
AVR Setup
==============

AVRマイコン関係のセットアップに関して記述していく. 基本的にはCLIで作業を行っていく想定. 

とはいえ, 基本的にはC言語で書き, ``avr-gcc`` を利用する. また, 開発環境はMacで, プラグインマネージャはhomebrewとする.

Install
==========

基本的には以下のサイトを参考にするといいだろう.

> homebrew_avr_

これで基本的なツールチェインは入るようだ.
