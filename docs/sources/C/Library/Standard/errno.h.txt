=========================
errno.h
=========================

'errno.h' の情報を記述する.

Variables
===========

::

    extern int errno

errnoは, 内部でグローバルに定義された整数値. errno.h内にはエラーに対応した定数が定義されており, エラーが起きた場合にerrnoに値を設定するシステムコールや関数がある.

何かしらの処理に失敗した場合, この値を確認することでエラーの種類を特定できる場合がある.

これを出力する方法として, `perror` が :doc:`./stdio.h` に定義されている.
