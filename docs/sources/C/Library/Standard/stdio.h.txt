Functions

perror
---------

::

    void perror(const char *str)

エラーメッセージ str と errno に対応するシステムエラーメッセージを標準エラー出力 (`stderr`) に出力する.

出力は 「str文字列: errnoに対応するシステムエラーメッセージ」となる. ただし, strにNULLを指定した場合と, 空文字列の場合には, 「errnoに対応するシステムエラーメッセージ」のみを出力する.
