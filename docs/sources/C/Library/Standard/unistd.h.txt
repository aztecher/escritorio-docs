=========================
unistd.h
=========================

'unistd.h' の情報を記述する.

Functions
===========

write
-------

::

    ssize_t write(int fd, const void *buf, size_t count);

bufが指すバッファから, ファイルディスクリプタが参照するファイルへ, 最大countバイトを書き込む. 書き込み対象の物理メディアサイズの領域がない場合や, リソース上限に達した場合など, 書き込み途中でシグナルハンドラに割り込まれた場合などは, 書き込まれるバイトはcountよりも小さくなる場合がある.

成功した場合, 書き込まれたバイト数が返され, エラーならば-1が返され, errnoが適切に設定される.


read
-------

::

    ssize_t read(int fd, void *buf, size_t count);

ファイルディスクリプタから最大countバイトをbufで始まるバッファへ読み込もうとする. 

成功した場合, 読み込んだバイト数を返す. ファイル位置はこの数だけ進められる. エラー時は-1が返され, errnoが適切に設定される.


close
--------

::

    int close(int fd)

ファイルディスクリプタをクローズする. そのディスクリプタはどのファイルも参照していない状態になり, 再利用が可能になる.

成功した場合は0を返す. エラーが発生した場合は-1を返して, errnoが適切に設定される.
