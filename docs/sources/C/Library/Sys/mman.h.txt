Functions
============

mmap
-----------

::

    void *mmap(
        void *addr,
        size_t lenght,
        int prot,
        int flags,
        int fd,
        off_t offset);

mmap()は呼び出し元のプロセスの仮想アドレスに対して, 新しいマッピングを作成する.
第一引数で, マッピングの開始アドレス(NULLの場合はカーネルがどのアドレスにマップするか決める), 第二引数で何バイトのデータをマップするか, 第5引数がマッピングの内容のファイルディスクリプタ, 第6引数がfdのどの位置(オフセット)からマッピングするかを指定する.
第3引数は, マッピングのメモリー保護をどうするかしていしたもの. 読み, 書き, 実行, アクセス拒否の選択(及び論理和)が選択できる.
第4引数は, 他のプロセスからみえるかどうかなどの細かいフラグを調整できる. (現時点では詳しく言及しない)

返り値としては, マップに成功するとマップされた領域へのポインターを返す. 失敗すると値 `MAP_FAILED` を返し, errno がエラーの内容に従ってセットされる.

munmap
-------------

::

    int munmap(void *addr, size_t n);

munmap() でプロセスのアドレス空間からメモリマップされた領域を取り除くことができる.
第一引数には, mmap()に成功した際に得られたアドレス(マップされた領域のアドレス)を与え, 第二引数にはマップされた領域の大きさを与える.
