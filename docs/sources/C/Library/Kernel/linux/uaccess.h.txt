=========================
uaccess.h
=========================

'linux/uaccess.h' の情報を記述する. ('asm/uaccess.h' の内容と見てもいい)

Functions
============

copy_to_user
--------------

::

    unsigned long copy_to_user(
        void __user *to
        const void* from,
        unsigned long n);

カーネル空間からユーザ空間へ(ブロック)データをコピーする
第一引数に, ユーザ空間のアドレス番号を指定(コピー先)
第二引数に, カーネル空間のソースアドレスを指定し(コピー元),
第三引数で何バイトcopyするかを指定する.

copy_from_user
-----------------

::

    unsigned long copy_from_user(
        void* to,
        const void __user *from,
        unsigned long n)

ユーザ空間からカーネル空間へ(ブロック)データをコピーする.
第一引数に, カーネル空間のアドレスを指定(コピー先),
第二引数に, ユーザ空間のソースアドレスを指定し(コピー元),
第三引数で何バイトcopyするかを指定する.

get_user
-----------

::

    get_user(x, ptr)

ユーザ空間からカーネル空間に対して単純な値をコピーする. 単純型(char, int)をサポートしており, 構造体やアレイなどの長いデータ型は対応していない. 第一引数に結果を保存する変数, 第二引数にユーザ空間のソースアドレス(コピーしたい値が入っているユーザ空間アドレス)を指定.

返り値は, 成功すると0, エラーの場合は `-EFAULT` を返す.

put_user
-----------

::

    put_user(x, ptr)

カーネル空間からユーザ空間に対して単純な値をコピーする. 単純型(char, int)をサポートしており, 構造体やアレイなどの長いデータ型は対応していない. 第一引数にユーザ空間にコピーする値を, 第二引数にユーザ空間のソースアドレス(コピー先のユーザ空間アドレス)を指定.

返り値は, 成功すると0, エラーの場合は `-EFAULT` を返す.
